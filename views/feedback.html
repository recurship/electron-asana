What does this app do?

This is a non-offical chrome app for Asana using their API. The core aim of this app is to allow the user fast and easy access to their tasks, and allow them to update its status, comment or attach files straight from desktop. It also syns stuff offline, so you can view your tasks with or without an internet connection. Finally it also as an auto-refresh built in, notifying you of changed tasks in the project / workspace you are currently active. We currently do not support editing tasks, deleting them or adding new ones and several other features. If you are interested in any of those features please check out 'There is a feature missing section'.

I cant understand how this works

The app has two main views - the task view and the settings view. You can switch to either of them using the tabs on top. The task view shows the current tasks in the currently selected project. You also have the option to see the tasks assigned to you (in a workspace) using the first option. Once you have selected your preferred project, you can set your view according to your liking - you can filter according to the completion status, assigned status or the person assigned (only if you are not using the workspace scope). If you have any further questions - please do not hesitate to contact us using the button on top.

There is a feature missing

The app was designed according to the interal requirements of the workflow at Recurship, thus it currently has a very basic subset of the features which were needed. However this app is under constant improvement, so if we get enough requests for a feature that we dont require, we are more than happy to implement it. So send in your requests! Use the button on top to get in touch with us. Currently the following set of features is missing:
  • Add tasks
  • Edit tasks (except for completion status)
  • Delete tasks
  • Reorder tasks
  • Move tasks between projects and workspaces
  • Heart tasks
  • Add followers to a task
  • Keyboard shortcuts
  • Tags
  • Tagging users in comments
  • Search
  • Drag and drop to attach files
  • Advanced filtering
  • Autorefresh subtasks

What is the best way to setup this app

We have been using Asana for a while now and have designed this app to fit our workflow. This may or may not be similar to how you use Asana.

Currently use user has tasks assigned to them over several projects. They use the 'All (assigned to me)' option to see all the tasks assigned to them at the moment. Most of use filter by tasks to do, and some of us use the Assignee status 'Today' to just see the really important tasks. Over the day if any of these tasks is updated, we get a notification. If we want to comment, or attach a file - it can be done using this as well. If you want to access the subtasks, we can simply click on the subtasks button in the detail view to dive into the subtasks list. Going back is as simple as clicking the back arrow on the top left.

Normally we keep the window open and in view in a corner of one of our monitors. However we can use the shortcut key cmd + shift + 5 to toggle the window really fast. The window is designed to stay on top of all applications so you can see your tasks at all times. This can be changed in the settings. You can also resize the window to just see one task so it takes minimal space.
